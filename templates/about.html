About

I made this website to improve my Python and web development skills. The goal of the project was to create a website that can build Spotify playists automatically based on a tracklist from 1001Tracklists.com It works by connecting to a users Spotify account through the Spotify Web API. Then scraping the given URL for all the tracks listed and creating a playlist of the ones available on Spotify.

While building this project I learned more about web-scraping with BeautifulSoup, data manipulation with Pandas, and the integration of the MongoDB client.
